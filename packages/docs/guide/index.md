# 介绍

Vue CRUD Tools 是一个专为 Vue 3 设计的组件库，帮助开发者快速构建 CRUD（创建、读取、更新、删除）界面。

## 为什么选择 Vue CRUD Tools？

- **Vue 3 原生支持**：充分利用 Vue 3 的 Composition API 和响应式系统
- **TypeScript 支持**：完整的类型定义，提供更好的开发体验
- **轻量级设计**：只包含必要的组件，避免过度抽象
- **易于定制**：基于 UnoCSS，支持灵活的样式定制
- **开箱即用**：提供常用的 CRUD 组件和工具函数

## 核心特性

### 🚀 现代化技术栈
- Vue 3 + TypeScript
- Vite 构建工具
- UnoCSS 原子化 CSS
- ESLint + Prettier 代码规范

### 📦 丰富的组件库
- CrudTable：数据表格组件
- EButton：增强按钮组件
- 更多组件持续开发中...

### 🛠️ 开发工具
- 类型安全的 API
- 完整的文档
- 示例代码
- 单元测试
